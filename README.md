  ```sh
  pip install -r requirements.txt
  ```
### Запуск обучения (модель детекции yolov8s) (внести изменения в cfg.yaml и smokers.yaml)
В smokers.yaml указать пути к датасету
  ```sh
  python train.py
  ```
